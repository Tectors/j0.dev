# 📖 About
**`j0.dev`** aims to elevate the existing **Local Fetch** workflow from **JsonAsAsset**, making it easier to set up builds and port assets.

# Updating Submodules
## CUE4Parse

To update CUE4Parse to the latest commit, do these commands in order:

```
cd Dependencies/CUE4Parse
git fetch
git checkout master
git pull
```

# Inspirations
**[Fortnite Porting](https://github.com/h4lfheart/FortnitePorting)**  
A beautifully clean and modern user interface, a huge inspiration for **`j0.dev`**'s design.  
Thank you to the creators for setting such a high bar.

**[FModel](https://github.com/4sval/FModel)**  
The original tool that defined Unreal Engine data exploration.  
I've used it extensively, and its developers created CUE4Parse, and helped deepen my understanding of CUE4Parse.
Sincere thanks to the team.
